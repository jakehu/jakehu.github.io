Ldap

轻型目录访问协议（英文：Lightweight Directory Access Protocol，缩写：LDAP，/ˈɛldæp/）是一个开放的，中立的，工业标准的应用协议，通过 IP 协议提供访问控制和维护分布式信息的目录信息。

摘自维基百科，关于 LDAP 概念和原理可以看看这篇文章 LDAP 概念和原理介绍

端口：默认服务端口使用 389 端口
