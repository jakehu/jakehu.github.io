使用

比如我们现在有一个场景，为 sanic 项目创建一个虚拟环境
第一步：
我们需要安装需要的 Python 版本比如 3.9.5
pyenv install 3.9.5
第二步：
创建一个 3.9.5 版本的虚拟环境
pyenv virtualenv 3.9.5 sanic
第三步：
去到 sanic 项目目录
pyenv local sanic
第四步
取消设定
pyenv local --unset
第五步
导出 requirements.txt
pip freeze > requirements.txt
